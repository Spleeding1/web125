What Is That?

At some point and time you have probably seen the image above. Usually referred to as the Check Engine Light, its technical name is the Malfunction Indicator Light, or MIL.

Different people will deal with it in different ways. If no noticeable running issue is felt, some might ignore it, or if they don’t want to look at it, put a piece of tape over it. Others will wait until the next service to investigate, or schedule an appointment immediately.

Depending where you live in the state of Pennsylvania, you will have to repair the problem to pass the annual emissions test. Having the MIL illuminated during testing will fail in any county where the On-Board Diagnostic 2(OBD-II) test is administered. There are 42 counties in the state that have a visual inspection performed with the annual safety inspection instead of an OBD-II test. Having the MIL illuminated is irrelevant for this type of inspection. It also does not matter in an OBD-II inspection county if the vehicle was driven less than 5,000 miles in the previous 12 months, as an exemption sticker will be issued.

If you don’t want to pay for diagnostic, you can stop at an auto parts store where they will read the diagnostic trouble code for free. Just beware that they might not have a solution, or the correct one (you get what you pay for). It should be investigated and resolved to keep your vehicle in top shape.

Problems that Will Light the MIL

  • Spark Plug
  • Fuel Injector
  • Oxygen Sensor
  • Low Oil Pressure
  • Vacuum Leak
  • Incorrect Valve Adjustment
  • Low Battery Voltage
  • DTC stored in another control module
    • Electric Power Steering
    • Anti-Lock Braking System
    • Safety Restraint System
    • All Wheel Drive System

When Is the MIL On?

There are different reasons why the MIL (Malfunction Indicator Light) illuminates, but the root cause is that a Diagnostic Trouble Code (DTC) has set in the Engine Control Module (ECM, sometimes called Powertrain Control Module, or PCM). A DTC is set when a control module detects an error.

There are three different classes of errors. The first class is a 2-trip error. If a problem is detected, it will store a temporary DTC but will not illuminate the MIL. If the same error occurs a second time, a permanent DTC will be stored and the MIL will illuminate.

The second is a 1-trip error. Just like it sounds, the first time the error occurs the MIL will illuminate.

The third is an error indicating damage to the catalytic converter could occur. Instead of the MIL just illuminating, it will flash. It is best to pull over and turn the vehicle off. Catalytic converters are expensive, and you probably don’t want to pay for one if you don’t have to.

Most control modules communicate with each other in vehicles today. If one sets a DTC, they all might. It is a redundency. If one module dies, you can still get information as to what is wrong.
